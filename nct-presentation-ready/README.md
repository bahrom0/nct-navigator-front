# NCT Navigator — готовая веб-презентация

Внутри находится готовая реализация маршрута `/presentation`:

- 6 полноэкранных слайдов;
- переходы без длинного скролла;
- управление стрелками, Space, PageUp/PageDown, Home/End;
- полноэкранный режим по клавише `F`;
- progress bar и номера слайдов;
- мобильные свайпы и колесо мыши;
- адаптация под 1366×768 и 1920×1080;
- настоящий QR-код на `https://mmtai.xyz`;
- тёмный и светлый варианты логотипа;
- готовые временные изображения, которые можно заменить своими.

Начните с файла [INSTALL.md](./INSTALL.md).
